# REACT-FRONTEND

# Quick Start

1. Make sure `NodeJS & Yarn` installed
2. Use VSCode with `eslint & prettier` extensions
3. Install dependencies with `$ yarn`
4. Run with `$ yarn start`

## prerequisite

```markdown
React | Redux | Redux-Saga | immer | TypeScript
```
